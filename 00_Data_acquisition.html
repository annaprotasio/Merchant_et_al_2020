RNAseq datasets

RNAseq data has previously been submitted in Bernot et al 2020. The sample descritpion ans accession numbers can be found in their Supplementary Table 1. Note that there accession numbers correspond to secondary sample accession while the ones used by us correspon to the run accession, which matched the fastq file path (see below). To aid samples matching, we used the filereport from ENA for this project to match all accession numbers and a summary of them is presented in the table below. Detailed sample descriptions including life cycle statge are missing from the ENA project filereport but was found by searching for individual sample accession pages (see example for Sample Accession SAMN08145872) and refering to the “Additional Attributes”
